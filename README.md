# PokéWorld

PokéWorld is a modern Pokédex web app built with React. Explore Pokémon information, types, habitats, and stats in a beautiful interface.

## Technologies Used

- **React**: Modern UI library
- **Styled Components**: Dynamic styling
- **SignalR**: Real-time updates
- **Context API**: Global state management
- **PokéAPI**: Pokémon data source

## Features

- Search Pokémon by name
- Filter by type and habitat
- Easy pagination
- Pokémon detail modal
- Login, registration, and account editing
- Account deletion with confirmation
- User profile and Pokémon collection page

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.

### Steps

1. Clone the repository:

```bash
git clone https://github.com/anilkumar/pokeworld.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the project:

```bash
npm start
```
